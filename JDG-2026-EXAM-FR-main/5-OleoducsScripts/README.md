# Mandat 5 – DevOps – Oléoducs Scripts

**Contexte**

Oléoducs Scripts est une entreprise spécialisée dans le développement des opérations et la gestion d’environnements Docker. Dans le cadre de ce mandat, votre équipe est mandatée pour relever une série de défis de DevOps liés à la gestion et à l’orchestration de conteneurs Docker.
