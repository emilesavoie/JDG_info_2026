# Q2 - Schéma SQL (4 points)

**Objectif :** En vous aidant de votre Diagramme Entité-Association, rédigez le schéma SQL dans `SCHEMA.sql`.

## Instructions

- Modifiez le fichier `SCHEMA.sql` dans ce répertoire
- Utilisez le langage MySQL si possible
- Implémentez toutes les tables identifiées dans votre diagramme Entité-Association
- Définissez les clés primaires et étrangères appropriées
- Respectez les bonnes pratiques de nommage

## Éléments attendus dans le schéma SQL

### Tables

- Table des entités principales
- Tables de liaison si nécessaire

### Clés

- Clés primaires pour chaque table
- Clés étrangères pour les relations entre les tables

### Types de données

- Choix appropriés des types de données
- Tailles de champs cohérentes
- Gestion des valeurs nullables

## Ressources

- [Documentation MySQL](https://dev.mysql.com/doc/)
- [Guide des diagrammes UML](https://www.uml.org/)
- [Bonnes pratiques de conception de bases de données](https://dev.mysql.com/doc/refman/8.0/en/database-use.html)
