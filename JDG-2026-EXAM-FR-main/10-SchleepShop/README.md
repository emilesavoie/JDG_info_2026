# Mandat 10 – AI - SchleepShop

**Contexte**

SchleepShop développe des systèmes d'intelligence artificielle visant à améliorer la qualité du sommeil des utilisateurs. La plateforme collecte des habitudes de sommeil, des données physiologiques et des préférences pour :

- Prédire les meilleures conditions de sommeil pour un utilisateur donné.
- Générer des plans de sommeil personnalisés.
- Détecter des anomalies ou tendances dans le comportement de sommeil.

Dans ce mandat, votre équipe est chargée de fournir des prompts efficaces pour les modèles d'IA utilisés par SchleepShop.
