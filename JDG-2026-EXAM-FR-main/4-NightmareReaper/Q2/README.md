# Q2 - Stéganographie (9 points)

**Objectif :** Extraire **3 flags** des fichiers fournis et fournir une **explication claire** de la démarche utilisée pour chaque extraction.

## Contexte

Trois fichiers contiennent chacun un élément caché. L’objectif est d’identifier une approche adaptée pour décrypter leur secret et de documenter une démarche reproductible.

## Instructions

- Résoudre les **3 problèmes** ci-dessous.
- Pour chaque problème : donner les **étapes**, l’outil (ou script) utilisé, et les paramètres si applicable.
- Fournir le **flag** correspondant.

### Exemple

Si vous avez trouvé le flag **"FLAG-HAHAHAHAH"** à l'aide d'un outil en ligne :

- Mentionnez les paramètres et l'outil utilisé.

Si vous avez utilisé un outil personnel :

- Décrivez la méthode en détail.

## Problèmes

### Problème 1 : Altérations

Le fichier **alterations.png** semble légèrement différent de son correspondant **non-altered-bonus.jpg**.  
Saurez-vous trouver la différence ?

### Problème 2 : Contenu Caché

Le fichier **InYourDreams.jpg** cache quelque chose d'intéressant.  
Qu'est-ce que cela pourrait être ?

### Problème 3 : Fichier Suspect

Le fichier **whats-this.mrx** semble louche.  
Qu'est-ce que c'est ?

## Contraintes techniques

- Le **bruteforce est permis** (c'est votre machine).
